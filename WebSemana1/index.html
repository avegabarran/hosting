Áreas de Conocimiento

El estudiante comprenderá el código HTML y su evolución a HTML5 y podrá utilizar de manera correcta las etiquetas. Entenderá la forma de aplicar e interactuar con las hojas de estilo en cascada (CSS3) de manera que permitan agilizar el proceso de diseño e implementación de un sitio web. Podrá aplicar eventos para elementos HTML con el lenguaje Javascript y podrá manipular la estructura y agregar efectos visuales con JQuery, así como consumir servicios web REST (REpresentational State Transfer) en formato de mensaje JSON y aplicación de estructura XML en los desarrollos WEB.

Entre otros conceptos cubiertos está, la creación de sitios responsivos para ser visibles desde cualquier dispositivo con el uso de media queries y frameworks como Bootstrap, y por último introducirá en el desarrollo de aplicaciones del lado del servidor con el lenguaje PHP y/o ASP.net.

Obtener y aplicar los conocimientos necesarios en las tecnologías que permiten diseñar, desarrollar y publicar sitios web.

  • Conocer los elementos y conceptos básicos que se involucran en la web para publicación de sitios.
  • Conocer el lenguaje HTML y su evolución a HTML5 para desarrollar sitios web.
  • Utilizar correctamente los selectores de estilo CSS3.
  • Conocer el lenguaje Javascript, para elaborar enlaces con eventos en elementos html.
  • Manipular la estructura de HTML y agregar efectos visuales con JQuery.
  • Desarrollar sitios web responsivos con el framework Bootstrap.
  • Consumir Servicios Web REST con formato JSON, con JavaScript y JQuery.
  • Aplicar programación con XML en los desarrollos web.
  • Conocer conceptos básicos del lenguaje PHP y/o ASP.net para desarrollo de aplicaciones del lado del servidor.
